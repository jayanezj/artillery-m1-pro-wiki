Este repositorio está asociado a una instancia de Wiki.js – https://artillerym1.codigojose.com – de tal forma que se pueda utilizar como respaldo y sistema de colaboración para las páginas.

## Contribución

- Si quieres mejorar o añadir la traducción a algún idioma:
  - Crea una rama ``feature/IDIOMA`` (por ejemplo: ``feature/it``).
  - Añade las páginas de traducidas.
  - Realiza un pull request para integrar los cambios.
- Si quieres añadir una nueva entrada:
  - Determina primero si vas a añadir una entrada de ``guía``, ``mejora`` o ``resolución de problemas``. Si quieres proponer una nueva categoría hazlo en la sección de discusiones.
  - Crea una rama ``feature/new/NUEVA-ENTRADA`` (por ejemplo: ``feature/new/bed-fix``)
  - Añade el contenido de la nueva entrada al menos en inglés (puedes añadirlo en otros idiomas también).
  - Realiza un pull request para integrar los cambios.
